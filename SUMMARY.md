## 🎨 Design System

### Colors
- **Primary Purple**: #667eea (Links, accents)
- **Secondary Purple**: #764ba2 (Gradients)
- **Dark Slate**: #2c3e50 (Header, footer, text)
- **Light Gray**: #f8f9fa (Backgrounds)
- **White**: #ffffff (Content areas)
- **Accent Blue**: #3498db (Hover states)
- **Success Green**: #27ae60 (Confirmations)

### Typography
- **Font Family**: Segoe UI, sans-serif
- **Heading Size**: 2.5rem (H2), 1.2rem (H3)
- **Body Size**: 1rem
- **Line Height**: 1.6

### Spacing
- **Container Width**: 1200px max
- **Section Padding**: 4rem top/bottom
- **Element Gap**: 1-2rem
- **Margin Bottom**: 1.5-2rem

## 🔧 Customization Quick Reference

### Change Colors
Search in CSS and replace:
- `#667eea` → Your primary color
- `#764ba2` → Your secondary color
- `#2c3e50` → Your dark color

### Change Fonts
In CSS (line 15):
```css
font-family: 'Your Font Name', sans-serif;
```

### Add Images
```html
<img src="images/your-image.jpg" alt="Description">
```

### Update Social Links
```html
<a href="https://linkedin.com/in/yourprofile" target="_blank">LinkedIn</a>
```

### Modify Skill Proficiency
```html
<div class="skill-bar" style="width: 85%;"></div>  <!-- Change percentage -->
```
