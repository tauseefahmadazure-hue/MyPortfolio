# My Portfolio

A modern, responsive portfolio website showcasing projects, skills, experience, and services.

## Features

### 🎨 Design & UI
- Modern gradient design with purple and blue color scheme
- Fully responsive layout (mobile, tablet, desktop)
- Smooth scrolling animations
- Interactive hover effects
- Mobile hamburger menu navigation

### 📱 Responsive Design
- Mobile-first approach
- Breakpoints for tablets and desktops
- Touch-friendly navigation
- Optimized for all screen sizes

### 📑 Sections Included

1. **Hero Section** - Eye-catching welcome banner with call-to-action
2. **About Me** - Personal introduction with key highlights
3. **Experience** - Timeline of work experience and roles
4. **Featured Projects** - Showcase of 6 major projects with technologies used
5. **Services** - 6 key services offered (Web Design, Frontend, Backend, Mobile, SEO, Performance)
6. **Skills** - Organized by categories (Frontend, Backend, Databases & Tools) with proficiency bars
7. **Testimonials** - Client feedback and reviews with 5-star ratings
8. **Latest Articles** - Blog section with recent articles
9. **Contact Form** - Detailed contact form with validation
10. **Contact Links** - Multiple ways to get in touch (email, LinkedIn, GitHub, etc.)

### 🎯 Interactive Features

- **Smooth Scrolling** - Navigation links scroll smoothly to sections
- **Active Navigation** - Nav links highlight based on current section
- **Scroll-to-Top Button** - Appears when scrolled down
- **Mobile Menu Toggle** - Hamburger menu for mobile devices
- **Scroll Animations** - Elements fade in and slide up as you scroll
- **Form Validation** - Contact form with success feedback
- **Dynamic Effects** - Hover effects on cards and buttons

### 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Grid and Flexbox
- **JavaScript (Vanilla)** - No dependencies, pure JS for interactivity
- **Responsive Design** - Mobile-first approach

## How to Use

1. **Open in Browser** - Simply open `index.html` in any modern web browser
2. **Customize Content** - Edit the text, links, and information in the HTML
3. **Update Links** - Replace placeholder links with your actual social media and contact information
4. **Add Images** - Replace project image placeholders with actual images
5. **Deploy** - Upload to a web hosting service or GitHub Pages

## Customization Guide

### Update Personal Information
- Change "My Portfolio" in the header
- Update about section with your bio
- Add your actual experience and projects
- Replace contact information (email, phone, location)
- Update social media links

### Modify Colors
The main color scheme uses:
- Primary Purple: `#667eea`
- Secondary Purple: `#764ba2`
- Dark Background: `#2c3e50`
- Light Background: `#f8f9fa`

To change colors, search and replace these hex codes in the `<style>` section.

### Add Project Images
Replace the project card image divs:
```html
<!-- Instead of: -->
<div class="project-card-image">Project Image</div>

<!-- Use: -->
<div class="project-card-image" style="background: url('image.jpg') center/cover;">
```

### Update Skills Proficiency
Modify the skill bar width to reflect your actual proficiency:
```html
<div class="skill-bar" style="width: 95%;"></div>  <!-- Change 95% -->
```
