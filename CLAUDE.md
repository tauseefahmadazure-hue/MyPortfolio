# My Portfolio - Project Documentation

## 📋 Quick Reference

A complete, responsive portfolio website with 12 sections, contact form, and smooth animations. Single HTML file with no external dependencies.

All documentation is in the `doc/` directory:

- **[doc/README.md](doc/README.md)** - Project overview, features, and browser support
- **[doc/QUICKSTART.md](doc/QUICKSTART.md)** - Get started in 5 minutes with step-by-step instructions
- **[doc/CUSTOMIZATION.md](doc/CUSTOMIZATION.md)** - Detailed guide to customize colors, fonts, content, and sections
- **[doc/ASSETS.md](doc/ASSETS.md)** - Image optimization, file sizes, and resource recommendations

## 🚀 Start Here

1. Open `index.html` in your browser to see the portfolio
2. Read [doc/QUICKSTART.md](doc/QUICKSTART.md) for immediate customization
3. Follow [doc/CUSTOMIZATION.md](doc/CUSTOMIZATION.md) for detailed changes
